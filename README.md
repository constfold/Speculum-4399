## 特性
- 无需浏览器（只需本地 Flash Player Standalone）
- 无需账号（也可以玩造梦西游等需要账号的游戏）
- 自动化嗅探下 SWF 主文件

## 使用说明
运行 `py -3 main.py` 获取帮助。

## 依赖
- Python3
- Java
- [FFDec_lib](https://github.com/jindrapetrik/jpexs-decompiler/tree/master/libsrc/ffdec_lib)
